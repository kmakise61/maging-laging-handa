# Disaster Preparedness Hub — Final Website

Your site is ready. This folder contains:
- index.html
- app.js

## Quick preview (no install)
- Double-click `index.html` to open it in a browser.

## Local server (recommended)
- Python 3: `python -m http.server 8080` then open http://localhost:8080

## Deploy fast (any of these)
- **GitHub Pages**: create a repo, push these files to the default branch, then enable Pages for the root (`/`). Your site will publish to a `github.io` URL.
- **Netlify**: drag-and-drop this folder into app.netlify.com → "Add new site".
- **Vercel**: import a new project, select this folder, and deploy.
- **Cloudflare Pages**: create a project and upload this folder.
